

---------------------------------------------------------------------------------------------
一、创建数据库
        a.使用sql语句创建
                语法: create database 数据库名;
                举例: 创建一个名为test1的数据库
                        create database test1;
        b.使用图形化界面创建
	        选择连接--右键选选择"New Database" 输入数据库名,点击ok即可
---------------------------------------------------------------------------------------------
二、查看数据
        a.使用sql语句查看
                语法: show create database ;
        b.查看已有数据库的创建信息
                语法: show create database 数据库名;
        c.使用图形化界面查看创建信息
	选择要查看的库名,右键-"Edit Database"
---------------------------------------------------------------------------------------------
三、修改数据库
        a.使用sql语句对数据库进行修改
                语法: alter database 数据库名 character set 新编码;
	举例: 修改test1 库的编码为gbk;
        b.图形化界面修改
	选中库名 右键 选中"Edit Database"
---------------------------------------------------------------------------------------------
四、删除数据库
        注意:数据库一旦删除,则数据库中的表和数据都会被一同删除
        a.使用SQL语句删除
                语法:drop database 库名;
	举例: 删除test1库
        b.使用图形化界面删除
                选中要删除的库名--右键--选择"Delete Database"
---------------------------------------------------------------------------------------------
