
数据库概述
一、数据库简介
	1.数据库管理技术发展
		人工管理
		文件管理
		数据库管理系统
	2.数据库相关的概念
		1. 数据 描述事物的符号 如 文本、文字、图片、音频、视频
		2. 数据库 车库 粮库 存储数据
		3. 数据库管理系统 对数据库中数据进行管理
		    流行的关系型数据库(oracle MySql DB2 SQL Server)
		4. 数据库应用程序
		    最终用户使用的应用程序,在数据库管理系统基础上,带有图形化界面的应用程序
		5.数据库管理员
		    对数据库系统操作的人员,主要负责对数据库进行运营与维护
		6.最终用户
		    对数据库应用程序的使用者
		7.数据库系统由如下内容构成
		    数据库 数据库管理系统 数据库应用程序 数据库管理员 最终用户
	3.什么是SQL语言
		SQL: 结构化查询语言简称
		a. DQL 用于对数据库					进行 查询  select from where
		b. DML 用于对数据 					进行 增加 删除 修改
		c. DDL 用于对数据库  中的对象(表、索引、视图、触发器、函数) 	进行 增加 删除 修改
		d. DCL 用于对数据库	的权限				进行 授予与回收
		e. TCL 用于对数据库   事物 				进行 开启 提交 回滚
二、MySQL简介
	1.数据库分类
		a.关系型数据库(二维表)
		      oracle
		            优点:  性能高 安全性高
		            缺点:  管理维护操作复杂 收费价格昂贵
		      MySQL
		            优点:  开源 体积小 速度快 成本低
		      DB2
		            优点:  稳定性高 安全性高
		            缺点:  使用繁琐 适合大型应用系统
		      SQL Server
		            优点: 功能全面 效率高
		            缺点: 只能在windows系统环境中运行

		b.非关系型数据库(NOSQL) not only sql
		      解决超大规模与高并发性      MongoDB Redis
